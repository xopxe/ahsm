Script run_to_dot.lua

Simple dot export script.

This script will export a hsm library to a dot file representation. If filename is provided the output will be written to a file. Otherwise the dot output will be writted to stdout.

Usage:

  • $ lua run_to_dot.lua <fsm.lua> [filename]

  • $ lua tools/run_to_dot.lua examples/composite.lua composite.dot
$ lua tools/run_to_dot.lua examples/composite.lua | dot -Tps -o composite.ps
$ lua tools/run_to_dot.lua examples/composite.lua | dot -Tpng | display -
