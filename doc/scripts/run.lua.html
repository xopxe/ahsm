Script run.lua

Simple run script.

This script will execute a state machine from a library. This script must be placed in the same directory with hsm.lua.

Usage:

  • lua run.lua  [debug] [forever] [sleep=none|os|socket] [time=ahsm|os|socket] <fsm.lua>

  • [debug] enables the ahsm logging using print.

  • [forever] do not not stop even if the machine becomes idle.

  • [sleep=none|os|socket] selects a sleeping mode.
"none" is polling, "os" uses a call to the sleep pogram,
and "socket" uses luasockets sleep().

  • [time=ahsm|os|socket] sets the ahsm.gettime function.
"ahsm" keeps it unchanged, "os" sets to call to os.time(),
and "socket" set to luasockets gettime()
